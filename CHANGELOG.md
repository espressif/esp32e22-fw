# Changelog

All notable changes to the firmware in this repository are documented in this
file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

Firmware in this repository uses a single monotonically increasing integer as
its version number, as described in [VERSIONING.md](VERSIONING.md). `version: 0`
is reserved as a non-official placeholder; official releases start at `1` and
are tagged in git as `v<version>` (for example, `v3`).
